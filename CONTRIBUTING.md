# Contributing to TRACE-AI

Thank you for your interest in contributing to TRACE-AI (Transparent Reporting for Agentic Catalysis Enabled by AI). This project aims to establish rigorous, reproducible, and auditable standards for AI-driven catalysis research.

We welcome contributions from the community across data, models, workflows, and reporting standards.

---

## Scope of Contributions

Contributions may include:

- Improvements to the TRACE-AI checklist
- New templates (e.g., agent cards, data manifests, experiment logs)
- Example workflows or case studies
- Documentation and clarity improvements
- Tools for validation, auditing, or visualization

---

## How to Contribute

### 1. Fork the repository
Create a personal fork of this repository.

### 2. Create a branch
git checkout -b feature/your-feature-name

### 3. Make your changes
Ensure:
- Clarity and reproducibility
- Alignment with TRACE-AI principles
- Minimal unnecessary complexity

### 4. Commit and push
git commit -m "Add: concise description of contribution"
git push origin feature/your-feature-name

### 5. Submit a Pull Request (PR)
- Clearly describe the motivation and contribution
- Reference related issues if applicable
- Provide examples where helpful

---

## Contribution Principles

All contributions should aim to:

- Enhance transparency in AI-enabled catalysis
- Enable reproducibility across labs and workflows
- Promote standardization without restricting innovation
- Support multimodal data integration
- Facilitate auditing and validation of agentic workflows

---

## Review Process

- Maintainers will review PRs for clarity, relevance, and consistency
- Feedback may be provided for revision
- Accepted contributions will be merged and attributed

---

## Code of Conduct

We expect all contributors to engage respectfully and constructively. TRACE-AI is a collaborative effort to advance the scientific community.

---

## Questions

If you are unsure where to start, please open an issue or discussion.

We appreciate your contributions to advancing transparent and reproducible AI-driven science.
